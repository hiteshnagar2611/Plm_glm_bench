2. VARIANT FILTERING PIPELINE
================================================================================

Step 1: Initial ClinVar Download
    Input:  All variants from ClinVar variant_summary.txt.gz
    Filter: Assembly = GRCh38
            Variant type = SNV (single nucleotide variant)
            Clinical significance = Pathogenic OR Benign
            (excluded: Likely Pathogenic, Likely Benign, Uncertain Significance,
             Conflicting Interpretations, Risk Allele, Drug Response, etc.)
            Review status >= 1 star (at least one submitters)
            Chromosome in {1-22, X} (standard chromosomes only)
            Not somatic
    Output: 25,162 variants

Step 2: Missense-Only Filtering
    Filter: Only variants with single amino acid changes (missense)
            Requires valid protein annotation (ref_aa, alt_aa, aa_position)
            Only one MANE transcript per gene (single-transcript genes)
    Output: 25,162 missense variants across 1,377 genes

Step 3: Gene Selection (200-Gene Subset)
    Criteria:
      - Must have >= 2 pathogenic AND >= 2 benign variants
      - Balanced representation (composite scoring)
      - Composite score = 0.6 * balance_score + 0.4 * count_score
        where balance_score = 1 - |n_pathogenic - n_benign| / (n_pathogenic + n_benign)
        and   count_score = min(n_pathogenic, n_benign) / max_across_all_genes
      - Top 200 genes by composite score selected
    Output: 10,224 variants across 200 genes

5. MODEL-SPECIFIC DATA PREPARATION
================================================================================

5.1 Protein Language Models (ESM2, SaProt, ProtT5)
    Input format: Protein amino acid sequences
    Variant representation:
      - Wildtype: Full protein sequence
      - Mutant: Full sequence with single amino acid substitution
    Scoring method: Log-likelihood ratio (LLR)
      LLR = log P(mutant | sequence) - log P(wildtype | sequence)
    Note: For ESM2 and SaProt, LLR is negated for evaluation
          (more negative LLR = more pathogenic → negate for standard AUROC)
    Variant count: ~2,435 variants (filtered by AA position <= 1022 for ESM2,
                    plus exclusion of stop codons and missing proteins)

5.2 Genome Language Models (AlphaGenome, NT-v2, HyenaDNA)
    Input format: 6,001 bp DNA context window
    Variant representation:
      - Wildtype: Full 6,001 bp context with reference allele
      - Mutant: Full 6,001 bp context with alternate allele
    Scoring method:
      - AlphaGenome: Delta-score (difference in 5 output modalities)
        RNA_SEQ, SPLICE_JUNCTIONS, SPLICE_SITES, SPLICE_SITE_USAGE, DNASE
        API minimum interval: 16,384 bp (extended context)
      - NT-v2: Log-likelihood ratio on 6-mer tokenized sequence
      - HyenaDNA: Log-likelihood ratio on character-level DNA tokens
    Variant count: 10,224 (all variants)

5.3 Structure-Aware Models (SaProt)
    Additional input: PDB 3D structure (Foldseek 3Di encoding)
    Sequence format: Interleaved AA + lowercase(3Di) codes
      Example: AA='GDK', 3Di='DVP' → SaProt input='GdDvKp'
    Low-confidence residues (pLDDT < threshold): masked with '#'
    Structure extraction: Foldseek structureto3didescriptor
    PDB coverage: 189 structures covering 199 genes
    Effective variant coverage: ~2,467 variants (25.6% of total)
      (most PDB structures don't cover full protein length)

8. EVALUATION METRICS
================================================================================

All metrics computed via scripts/11_evaluate.py:

- AUROC: Area Under the Receiver Operating Characteristic curve
  (standard metric for binary classification, threshold-independent)

- AUPRC: Area Under the Precision-Recall Curve
  (more informative for imbalanced datasets)

- Spearman r: Spearman rank correlation between model score and label
  (measures monotonic relationship)

- MCC: Matthews Correlation Coefficient
  (balanced measure even with class imbalance, binarized at score=0)

- Recall: Sensitivity for pathogenic variants
  (fraction of true pathogenic correctly identified at threshold=0)

Note on score direction:
  - Protein LMs (ESM2, SaProt): LLR = log P(mut) - log P(wt)
    More negative = more pathogenic → negated for evaluation
  - Genome LMs (AlphaGenome): Delta-score (higher = more different from ref)
  - NT-v2: LLR (direction depends on model training)

================================================================================
10. KNOWN LIMITATIONS
================================================================================

1. Class imbalance: 75.2% pathogenic vs 24.8% benign
   (reflects ClinVar submission bias)

2. PDB structure coverage: Only 25.6% of variants have their mutation
   position within a resolved PDB structure (for SaProt)

3. ESM2 length limit: Max 1022 amino acids → 4,087 variants beyond
   protein length excluded from ESM2/SaProt scoring

4. Stop codon variants: 3,669 variants with Ter (stop codon) as alt_aa
   excluded from protein LM scoring

5. Single-transcript bias: Only MANE Select transcripts used; some genes
   have multiple isoforms with different pathogenicity profiles

6. Review stars: Minimum 1 star required (at least one ClinVar submitter);
   higher stars (2-3) indicate more concordant submissions

7. PDB structure quality: PDB files may represent specific experimental
   conditions (crystal structure, cryo-EM) that don't reflect all
   physiological states

8. PDB residue numbering: SaProt scoring is limited by discrepancies
   between PDB residue numbering and ClinVar annotation. Only 322
   variants (3.1%) had matching reference amino acids in the PDB
   structure at the expected position. This is a known limitation of
   using experimentally-determined structures for variant effect prediction.
